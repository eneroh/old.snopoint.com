    Overview
    This device has been receiving high praise for its upgradability, durability and capability even though it is a laptop from Late 2013.
    I purchased this laptop from ebay for $300 and decked it out with necessary upgrades to make it very formidable compared to recent laptops.
    The Intel i7 4702MQ that I installed into this unit performs slightly better than my Dell XPS's cpu in benchmarks (see below)

    image

    How does a cpu from 2014 slightly outperform a cpu from 2017?
    This is because the i7 4702MQ is a mobile processor. While the 8550U is an Ultra low power cpu. Meaning some "cpu power" is sacrificed to improve the laptop's battery life. If you compare tdps of the CPUs this makes this point clear.

    I also achieve 4 hours of battery life on my T440P compared to my Dell XPS's 12+ hours

    Router

    [images]


    Usage:
    Fairly self-explanatory. Handles network routing of traffic. Where I port-forward, handle security, rarely run services, monitor traffic.

    Overview
    Networking can be very tricky but once you get the hang of it, it's interesting. As the router I purchased doesn't have modem functionality, I still use my old modem router but in bridging mode, which puts the device into a low power state and acts strictly as a modem connected to my new router via an ethernet cable connected to the ethernet and wan ports respectively.

    Netgear Nighthawk R8000 AC3200

    I will not be listing this devices specs, as the specs weren't the focus. This router was typically $300 at it's release, however I managed to wrangle one for $120. The main reason I had my eyes on this router was because of it's open source firmware support, what's better is that the firmware is still being updated today. The original firmware Netgear Genie was replaced with FreshTomato. However the firmware is limited to certain cpu's, making a strict list of supported routers.

    Regardless, I am very impressed with this router so far. Down the track, I plan to use a full system with pfsense. But currently, I am happy with FreshTomato for home use.

    Smartphone

    [images]

    Usage:
    Make calls, send texts, device to test projects, use apps etc.

    Overview:
    A slight side project, after being an android user for many years. I decided to branch out and look into alternatives. Open source alternatives to be exact. LineageOS. A fork of CyanogenMod, which was a fork of android. LineageOS is a much more capable OS with support for a wide range of devices and added security measures missing within Google's Android. The custom rom flashing process wasn't too complicated and made the hands-on experience more enjoyable.
